# CodeAlpha Calculator

## Description
This is a simple and responsive calculator developed using HTML, CSS, and JavaScript. It performs basic arithmetic operations with a clean and user-friendly interface.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Clear Screen (AC)
- Responsive Design

## Technologies Used
- HTML
- CSS
- JavaScript

## Working
1. Open the calculator in your web browser.
2. Click the number buttons to enter values.
3. Select an arithmetic operator (+, -, ×, ÷).
4. Enter the second number.
5. Press the "=" button to display the result.
6. Use the "AC" button to clear the screen and start a new calculation.
